RIAEasyTM是一个在线可视化的单页 Web 应用 (single-page application 简称为 SPA)设计平台。 RIAEasyTM旨在实现RIA/SPA应用的快速、高质量开发，实现在线无缝可视化设计，实现模块化开发，实现移动、桌面系统统一的跨浏览器开发。 RIAEasyTM基于dojo构建（dojo 1.10），支持HTML5。 RIAEasyTM的进一步发展目标： 实现跨平台、无差别的桌面、移动应用设计； 实现云端设计、打包、运行、应用； 实现团队开发模式； 实现Design easy, Design anywhere 。 RIAEasyTM是面向跨平台的单页应用设计平台，与传统的网页设计模式差别较大，反而更接近传统的C/S桌面应用设计模式。 尽管RIAEasyTM也可以用来快速设计传统的网页，但这显然不是其真正的优势。 正如RIAEasyTM的名称已经表明的，这是一个用来做RIA的工具。如果您做过C/S桌面应用，用过Delphi、C++Builder、VisualStudio这些工具，那么就更容易理解RIAEasyTM。
